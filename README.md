# Sliding Puzzle

A Sliding Puzzle written in Rust for terminal.

## Features

- [x] A board of any size
  - [ ] Selectable at runtime
- [x] Multiple grid drawing styles
  - [ ] Selectable at runtime
- [x] Field shuffler
  - [x] Parity shuffling
- [x] Player input
  - [x] WASD
  - [x] Arrow keys
  - [x] Numpad
- [ ] Multiple end states
- [ ] Add structs or something, for better abstraction
